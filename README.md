# Mikaboshi

## 概要
Mikaboshiは、ネットワークトラフィックを可視化するアプリケーションです。
ブラウザ上から対象機器へのトラフィックを、リアルタイムで表示します。
表示は3Dサイバー空間のような、美しいUIで可視化します。

## 構成

### Mikaboshi-Agent
- トラフィックをキャプチャし、Mikaboshi-Serverに送信します
- トラフィックのキャプチャは、libpcapを使用します
- rustで実装されています

### Mikaboshi-Server
- Mikaboshi-Agentから送信されたトラフィック情報を受け取り、ブラウザ上に表示します
- トラフィック情報は、gRPCを使用して送信されます
- バックエンドは、rust言語で実装されています
- フロントエンドは、Reactで実装されています
- 3D空間のレンダリングは、Three.jsを使用します

## 機能

### リアルタイムトラフィック表示
- 最初にエージェントを選択し、そのエージェントに対するトラフィックを表示します
- エージェントは光る球体として表示されます
- エージェントの接続相手(Peer)は、光るディスクとして表示されます
    - 接続相手のIPアドレスは、ディスク内に出力されます
    - エージェントの球体を取り囲むように、Peerのディスクが複数配置されます
- エージェントと、Peerとの間のトラフィックは、光る線として表示されます
    - トラフィックのデータサイズは、線の色に反映されます


### ブラウザでのアクセス
ブラウザで `http://localhost:8080` にアクセスしてください。
